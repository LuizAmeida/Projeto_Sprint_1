# 🐍 Sprint 1 - Fundamentos de Python para Análise de Dados

## 📌 Visão Geral

Este projeto faz parte do **Bootcamp Analista de Dados da TripleTen** (Sprint 1).  
O objetivo é aplicar **conceitos fundamentais de Python** para manipular, limpar e extrair informações de dados de clientes de uma empresa de e-commerce (Store 1).

## 🎯 Contexto do Projeto

A empresa Store 1 começou a coletar dados de clientes, mas os dados vêm em formatos inconsistentes. Minha missão foi utilizar **Python puro** (sem bibliotecas externas) para:

- Identificar problemas em variáveis (tipos, formatação)
- Limpar e transformar strings
- Converter tipos de dados
- Tratar erros com `try/except`
- Usar estruturas de repetição (`for`, `while`)
- Trabalhar com listas e listas aninhadas
- Calcular métricas simples (total, min, max)
- Filtrar dados com condicionais

## 🛠️ Tecnologias

- **Python 3.9+** (apenas biblioteca padrão)
- **Jupyter Notebook**
- Módulo `random` (simulação de compras)

## 📊 Tarefas Realizadas

| Tarefa | Conceito praticado |
|--------|-------------------|
| 1 | Limpeza de strings com `strip()` e `replace()` |
| 2 | Divisão de strings com `split()` |
| 3 | Conversão de tipos (`float` → `int`) |
| 4 | Tratamento de exceções com `try/except` |
| 5 | Loops `for` e conversão para minúsculas |
| 6 | Funções `sum()`, `min()`, `max()` |
| 7 | Loop `while` com `randint()` |
| 8 | Formatação de strings com f-strings |
| 9 | Soma de gastos em lista aninhada |
| 10 | Filtro por idade (< 30 anos) |
| 11 | Filtro combinado (idade + gasto > 1000) |
| 12 | Filtro por categoria ("clothes") |

## 📈 Resultados

- Clientes com menos de 30 anos e gastos > $1.000: `samantha` e `james`
- Clientes que compraram roupas (`clothes`): 5 clientes identificados
- Domínio de estruturas básicas de Python aplicadas a dados reais
